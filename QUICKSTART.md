## ğŸ› Troubleshooting

### Voice Not Working

- Try Chrome or Edge (best Web Speech API support)
- Check browser permissions for audio
- Click page before expecting audio (browser security)

### No Sound at All

- Check system volume
- Allow audio in browser
- Click somewhere on page first
- Try refreshing

### Interface Not Showing

- Check console for errors (F12)
- Clear browser cache
- Refresh page
- Make sure JavaScript is enabled

### Objects Not Appearing

- Make sure canvas isn't blocked
- Check if WebGL is enabled
- Try different dream descriptions
- Refresh and try again
